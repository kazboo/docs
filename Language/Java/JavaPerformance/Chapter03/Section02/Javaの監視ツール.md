# 3.2 Javaの監視ツール

* JVM自体に関する情報を得るには、Java向けの監視ツールが必要
  * jcmd
  * jconsole
  * jhat
  * jmap
  * jinfo
  * jstack
  * jstat
  * jvisualvm

## jconsole

* JVMのアクティビティをグラフィカルに表現する

* スレッドやクラスの利用状況、GCの内容などがわかる
